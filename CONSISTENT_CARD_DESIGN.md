## 🔧 جزئیات تکنیکی

### بخش هدر:
```
┌──────────────────────────────┐
│  • اگر رند: نمایش "رند" و ستاره‌ها
│  • اگر عادی: نمایش "سیمکارت عادی"
│  • ارتفاع ثابت: min-h-[80px]
│  • خلفیه: gray-50 (light) / gray-700 (dark)
│  • border-bottom برای جداسازی
└──────────────────────────────┘
```

### CSS کلاس‌ها:
- ✅ `bg-gray-50 dark:bg-gray-700/50` - خلفیه هدر
- ✅ `border-b border-gray-200 dark:border-gray-600` - خط جداکننده
- ✅ `min-h-[80px]` - ارتفاع ثابت
- ✅ `rounded-b-xl` - گوشه‌های قوسی (فقط پایین)
